## 🚀 Sobre o Projeto
O **StelarBank** é uma plataforma de gestão financeira e bancária digital desenvolvida para oferecer uma experiência de usuário arrojada e eficiente. O sistema foi construído pensando na escalabilidade e na modernidade das interfaces financeiras atuais.

## 🎨 A Grande Mudança Visual: Neo-Brutalismo
Recentemente, o projeto passou por uma reformulação visual completa. Saímos de um design convencional para adotar uma estética **Neo-Brutalista**.

### Por que mudamos?
1. **Identidade Forte:** O mercado financeiro é saturado de designs "clean" e minimalistas (estilo Nubank/Inter). Queríamos que o StelarBank se destacasse com uma personalidade vibrante e inesquecível.
2. **Clareza e Contraste:** O Neo-Brutalismo utiliza bordas grossas, sombras marcadas e cores sólidas que facilitam a hierarquia visual e a legibilidade de dados complexos (como extratos e transações).
3. **Experiência Arrojada:** Queremos que o usuário sinta que está usando uma ferramenta poderosa e moderna, fugindo do óbvio.

## 🏗️ Estrutura do Projeto
O repositório está organizado em uma arquitetura de monorepo simplificada:

### 📂 `bank/client` (Frontend)
- **Tecnologias:** React.js / Styled Components.
- **Destaque:** Implementação do `BrutalistLayout`, um sistema de componentes desenhado do zero para refletir a nova identidade visual.
- **Componentização:** Sidebar interativa, widgets de saldo, lista de transações otimizada e formulários de login com validação em tempo real.

### 📂 `bank/server` (Backend)
- **Tecnologias:** Node.js / Express / Supabase (Database & Auth).
- **Funcionalidades:** Gerenciamento de usuários, processamento de transações, integração com sistemas de pagamento e segurança de dados.
